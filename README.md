# 🧠 AI Document Intelligence

Convierte facturas físicas y documentos logísticos en **datos confiables** usando **IA multimodal (visión)**.

Este proyecto es un **prototipo funcional** desarrollado como parte de un *speed run* de producto, enfocado en resolver el problema real de los **datos sucios** en logística y operaciones.

---

## 🖥️ Funcionalidades principales

- 📤 Carga de imágenes (facturas / remitos)
- 👁️ Análisis con IA de visión
- 📊 Extracción estructurada de datos:
  - Proveedor
  - Fecha
  - Número de factura
  - Total
  - Moneda
- 📈 Indicador de confianza por campo
- ✏️ Edición manual (Human-in-the-loop)
- 💾 Guardado de datos confirmados en base de datos
- ✅ Flujo completo de validación antes de persistencia
